Como usar:

  • Digite ou cole seu texto na área de entrada e clique em "Processar Texto"
  • Arraste as palavras para reposicioná-las na tela
  • Clique em uma palavra para selecioná-la
  • Use as setas do teclado para mover palavras selecionadas
  • Pressione Delete para remover palavras selecionadas
  • Clique nas cores para alterar a cor das palavras
  • Use a roda do mouse para aumentar/diminuir o tamanho das palavras selecionadas
